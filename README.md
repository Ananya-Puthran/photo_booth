
# 📸 Aesthetic Photobooth Web App
🌐 **Live Demo:** https://ananya-puthran.github.io/photo_booth/  <br><br>
A cute and aesthetic **web-based photobooth** that captures **4 photos automatically**, applies **filters**, allows **stickers and frame customization**, and generates a **downloadable photo strip** — just like Korean photobooths.

Designed with a **girlish pastel UI** (pink themed) and built using **HTML, CSS, and JavaScript with the browser camera API**.

---

## ✨ Features

* 📷 **Webcam Photo Capture**

  * Uses laptop webcam via the **getUserMedia API**

* ⏳ **Automatic Photo Capture**

  * Takes **4 photos automatically**
  * **5-second gap** between each shot for posing

* ⏱ **Countdown Timer**

  * Displays **3…2…1 countdown** before each capture

* 🎨 **Photo Filters**

  * No Filter
  * Digicam
  * Vintage
  * Grayscale
  * Sepia
  * Soft Pink
  * Bright Kawaii

* 💖 **Cute Stickers**

  * Drag and drop stickers
  * Resize and rotate
  * Remove stickers

* 🖼 **Customizable Frames**

  * Change photo strip frame colors
  * Pastel themes like:

    * Pink
    * Lavender
    * Mint
    * Cream

* ✏ **Custom Caption**

  * Add a **short text message** at the bottom of the photo strip

* ⬇ **Download Final Photo Strip**

  * Generates a **vertical photobooth strip**
  * Download as **PNG image**

* 🎀 **Aesthetic UI**

  * Soft pastel theme
  * Rounded buttons
  * Cute UI animations
  * Mobile responsive

---

## 🖼 How It Works

1. Open the website.
2. Allow **camera access**.
3. Choose a **filter**.
4. Click **Start Capture**.
5. The system will:

   * Show a **3-2-1 countdown**
   * Take **4 photos automatically**
6. Add **stickers, frame color, and caption**.
7. Download your **photobooth strip**.

---

## 🛠 Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript**
* **WebRTC getUserMedia API** (camera access)
* **Canvas API** (image processing and final strip generation)
